Lumi Banking App

Android App

Lumi is a banking app optimized with accessibility features for visually impaired friends. With a user interface designed to be as simple as possible, we aim to address the needs and concerns of visually impaired users in carrying out their banking transactions.

Lumi is part of a larger project that encompasses both web and mobile application development. Lumi App is an Android banking application with key features including transfers, transaction history, and QR code scanning. Lumi is developed by a collaborative team consisting of a Project Manager, QA, UI/UX Designer, Front-End & Back-End Developers, and Android Developers.

Tech Stack

  • Android Jetpack
  • Retrofit
  • CameraX
  • Koin
  • Model View View Model (MVVM)
  • ZXing
  • etc.
